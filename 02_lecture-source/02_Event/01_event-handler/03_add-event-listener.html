03_addEventListener 방식

        EventTarget.prototype.addEventListener 메소드를 사용하여 등록 가능
        addEventListener('이벤트타입' , function [,useCapture]);
        첫 번째 매개변수는 이벤트의 타입을 지정해준다 
        두 번째 매개변수는 이벤트핸들러(함수)
        마지막 매개변수는 이벤트 전파단계(캡쳐링 or 버블링) 지정 (나중에 배울것이다)
        이 방식(addEventListener)은 이벤트 핸들러 프로퍼티에 바인된 이벤 핸들러에
        아무 영향을 주지 않는다
        동일한 HTML 요소에서 발생한 동일한 이벤트에 대해 하나이상의 이벤트 핸들러를
        등록 할 수 있으며, 이때 이벤트 핸들러는 등록 순서대로 호출 된다
