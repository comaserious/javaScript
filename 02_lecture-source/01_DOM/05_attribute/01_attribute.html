01. attribute

        Element.prototype.attributes 프로퍼티는 모든 어트리뷰트 노드를
        취득할 수 있다
        attributes 프로퍼티는 getter 만 존재하는 읽기 전용 접근자이다
        프로퍼티로 NameNodeMap 객체를 반환한다
    
        Element.prototype.getAttribute/setAttribute 메소드를 사용하면
        위에서 했던 attributes 프로퍼티를 통하지 않고
        요소노드에서 메소드를 통해 직접 HTML attribute 값을 취득하거나
        변경할 수 있어 편리하다
    
        Element.prototype.hasAttribute(attributeName)
        메소드를 사용하면 특정 HTML attribute 가 존재하는지 확인할 수 있다
        Element.prototype.removeAttribute(attributeName)
