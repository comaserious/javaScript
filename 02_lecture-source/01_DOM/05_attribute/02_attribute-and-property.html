02. HTML attribute 와 DOM 프로퍼티

        HTML attribute : HTML 요소의 초기 상태를 지정하면 변하지 않음
        attribute 노드에서 관리되면 값을 얻거나 변경하려면
        getAttribute/ setAttribute 메소드를 사용한다

        DOM 프로퍼티 : 사용자가 입력한 최신 상태를 관리한다
        사용자의 입력에 의한 상태 변화에 반응하여 언제나 최신 상태를 유지한다
    

        getAttribute 메소드로 취득한 attribute 값은 언제나 문자열
        하지만 DOM 프로퍼티로 취득한 최신 상태의 값은 문자열이 아닐 수 있다
        checkbox 요소의 checked attribute 값은 문자열이지만,
        checked 프로퍼티 값은 boolean 타입이다
