03. class를 이용한 요소노드 획득

        Document.prototype/Element.prototype.getElementsByClassName
        메소드는 인수로 전달한 class 어트리뷰트 값을 갖는 모든 요소 노드들을
        탐색하여 반환한다. 인수로 전달할 class 값은 공백으로 구분하여
        여러개의 class 를 지정할 수 있다
        메소드 이름의 Elements 가 복수형인 것에서 알수 있듯
        여러개의 요소 노드 객체를 갖는 DOM 컬렉션 객체인 HTMLCollection 을 반환
